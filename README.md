## 2、技术框架

运行系统：Windows

编程语言：Java

系统架构：B/S

前端框架：Vue+HTML+JavaScript+Css+LayUI

后端框架：SSM

数据库：MySQL

是否前后端分离：否

Maven项目：是

数据库表数量：15

运行环境：JDK8+MySQL5.7+idea+Tomcat8.5
